Pathfinding algorithm

Hold shift and hover over the bricks you want to select.
Press "s" and then select start point.
Press "e" and then select end point.
Press "d" and then select the brick you want to delete.
My first Pathfinding algorithm:
This is my first ever pathfinding algorithm and is just made for fun. It has a lot of bugs and you can 100% break it, but it does find the path in most of the easy to medium challenging mazes that you draw. Feel free to browse the code on my GitHub and don't hesitate to open an issue and suggest a change. This is based on "if/else if" so it crashes easily!
